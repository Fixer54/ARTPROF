1. Installation


Before the begin install "Video Gallery Addon" you should have to puchased, installed and activated Visual Composer and Video Gallery Addon. Because our addons will be works in Visual Composer. The first thing you need to do when you want to install plugin is to login to your site admin panel.

Once there, go to Appearance -> Plugins. section and install you purchased and downloaded Video Gallery Addon for Visual Composer WordPress Plugin via plugin installer.

Then you should activate the Video Gallery Addon

2. Usage

Video Gallery Addon have very simple usage. When create a page with Visual Composer, click to add element icon "+"



You'll see the "VC Video Gallery" button under the Elegant Mega Addons tab.



When click to the "Video Gallery" icon, "Video Gallery Settings" popup will opened and you will be able to define options.





Then you'll be able to add sub element into your page and you'll see the sub element settings.



When click to to add sub element icon you'll see the element settings within the popup window. You can define setting in this popup screen.
